# FurHope 🐾

FurHope is a comprehensive pet-care platform designed to connect pet owners, adopters, shelters, and animal lovers in one place. The platform provides pet adoption services, pet hotel reservations, an e-commerce marketplace for pet products, and a social media space dedicated to pets.

## Project Overview

FurHope was developed as a university group project with the goal of creating an all-in-one digital ecosystem for pet care and animal welfare.

## Project Evolution

The project was developed in two phases:

### Phase 1 – Web Application (Symfony)

The first version of FurHope was developed as a web application using Symfony and PostgreSQL. This phase focused on designing the platform's business logic, database structure, and core functionalities.

### Phase 2 – Desktop Application (JavaFX)

After completing the web version, the application was redesigned and rebuilt as a desktop application using JavaFX while preserving and extending the original features.

This second phase allowed us to improve the user experience and explore desktop application development.

## Main Features

### 🐶 Pet Adoption
- Browse pets available for adoption
- View pet profiles and information
- Submit adoption requests
- Manage adoption applications

### 🏨 Pet Hotel Booking
- Search pet hotels
- View available accommodations
- Book stays for pets
- Manage reservations

### 🩺 Veterinary Services
- Veterinarians can create professional profiles
- Veterinarians can publish their availability schedules
- Pet owners can browse available veterinarians
- Book veterinary appointments
- Manage consultation reservations

### 🛍️ Pet Marketplace
- Browse pet products
- View product details
- Manage product listings
- Purchase pet supplies and accessories

### 📸 Pet Social Feed
- Create and share pet posts
- Upload pet photos
- Interact with community content
- Follow pet-related updates and activities
